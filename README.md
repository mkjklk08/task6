# 🎮 Cyber-Tic-Tac-Toe (Multiplayer Engine)

A futuristic, real-time multiplayer Tic-Tac-Toe platform built with **MERN** stack and **WebSockets**. No registration required—just enter your name and start dominate the grid.

## 🚀 Features
* **Real-time Gameplay:** Zero latency moves using Socket.io.
* **Global Lobby:** Create your own game session or join an existing one.
* **Persistent Stats:** All wins are saved to MongoDB under your username.
* **Cyberpunk UI:** High-end dark theme with neon glowing effects and smooth animations.
* **Smart Logic:** Server-side win validation and draw detection.

## 🛠 Tech Stack
- **Frontend:** React.js, CSS3 (Glassmorphism & Neon Design)
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Real-time:** Socket.io

## 📦 Installation & Setup
Setup Backend:
- cd server
- npm install
- node server.js

Setup Frontend:
- cd client
- npm install
- npm start
